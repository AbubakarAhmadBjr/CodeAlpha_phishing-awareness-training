# Phishing Awareness Training

An interactive training application to educate users about phishing attacks, social engineering, and best practices for staying safe online.

## Features
- 📚 Comprehensive learning modules (psychology, real costs, red flags, incident response)
- 🧠 10‑question interactive quiz with instant feedback
- 🎨 Tap‑to‑expand cards for deep dives into each topic
- 🔍 Real‑world email and website examples
- 🛡️ Built with React, TypeScript, Tailwind CSS, and Vite

## Installation

```bash
git clone https://github.com/AbubakarAhmadBjr/CodeAlpha_phishing-awareness-training.git
cd CodeAlpha_phishing-awareness-training
npm install
npm run dev
Build for Production
bash
npm run build
The output is a single HTML file (dist/index.html) – ready to deploy anywhere.
